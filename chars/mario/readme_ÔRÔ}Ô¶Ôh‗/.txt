2008/8/7
元ネタ[マリオワールド　チビマリオ（SFC)を元に手書き]。
ボイスはスマブラ64と切り札はＸ

本家スマブラでは基準となる感じのポジションキャラだけど
MUGENでは、そうでもない｡マントとポンプで射撃or接近両極端なキャラには
かなり優位に戦える。
例にならってAIより手操作の方が強い


新しく「急降下」を取り入れてみた発動は「↓（空中のみ）」で通常より早く降下できます。
マリオじゃあんまし効果ないがメタナイトあたりに実装すればさらにコンボが決まりやすくなりそう。


完成度80%


-------------------[　弱攻撃　]-----------------------

ジャブ → ストレート →　けりあげ　　a（Hitで３回まで連続攻撃可）

-------------------[　強攻撃（方向キーは押したままでOK）　]-------

ミドルキック　　　　　→a

アッパーカット　　　　↑a（シールドキャンセル可）

マリオトルネード　　　↓a (→,←方向キーで微移動可）

-------------------[ スマッシュ攻撃（方向キーは早く離す）　]-------

ファイア掌底　　　　　　→a （溜め可）

スマッシュヘッドバッド　↑a （溜め可）（シールドキャンセル可）

ダブルキック　　　　　　↓a

-------------------[　空中攻撃（方向キーは押したままでOK）　]------

マリオキック　　　　　　a 

ドロップキック	　　　 ←a

メテオナックル	　　　 →a

エアスラッシュ	　　　　↑a　(→,←方向キーで微移動可）

マリオトルネード　　　　↓a (→,←方向キーで移動可）

-------------------[　投げ技　]----------------------------------

掴み　　　　　　　　　　z（ガードキャンセル可

掴み中コマンド

掴攻撃：つかみヘッドバット     a

前投げ：ヒコーキ投げ　　　　　 →

後投げ：ジャイアントスイング 　←

上投げ：スルーアップ 　　　　　↑

下投げ：スルーダウン　　　　　 ↓



-------------------[ 特殊技　]----------------------------------

ジャストガード　　　　　c（相手が近くにいる状態で相手の攻撃ギリギリでガードで発動）
　　　　　　　　　　 　　（効果はブロッキングとほぼ同じ）

シールド　　　　　　　　c (空中、しゃがみ、地上OK）

前転			→（シールド中に）

後転			←（シールド中に）

その場回避	　　　　↓（シールド中に）

空中回避　　　　　　　　c（空中のみ）

受身			c（くらい動作中限定　＋　地上近く）

急降下			↓（空中のみ）（通常より早く降下できる）

空中移動　　　　　　　　（空中にいる時の方向キーでかなり軌道修正ができる）

右上(1Pの時）に表示されているシールドゲージは c を押した状態だと減少し
それ以外の時には回復する。
シールドゲージが 0 になるとシールドが壊れ一定時間硬直状態になる。
