----------------------------------------------------------------------------------------
◆はじめに
　ダウンロード頂き有難う御座います。このキャラはCAPCOMのアーケードゲーム
　"ALIEN vs PREDATOR"のプレイヤーキャラの一人、"プレデターウォリアー"
（以下ウォリアー）を元に製作しました。
　
◆操作タイプや技の使用制限の変更
　"-2.cns"をメモ帳等で開き説明に沿って処理して下さい。
　
◆隠し
　キャラセレクト時にスタートを押しながら決定すると技がプレデターハンター版になります。

---------------------------★基本動作★-------------------------------------------------
★スピアまたはディスクを使う技はその武器を所持していないと使用できません。


【大ジャンプ】
         　　U + z

【フロントorバックジャンプ】
             F or B + z
　　   　
【バックステップ】
             z,B
　　     
【三角跳び】
　　　　(空中)画面端で壁の反対方向にレバーを入れながら z
　　　　　　　　　※レバー上要素入力でハイジャンプ。


【ガードキャンセル足払い】("A" or "B" type のみ)
　　(ガード中)　F + (x+y or x+z or y+z)

　　　　　　　           　※Power >= 500

---------------------------★特殊系★-------------------------------------------------
【レーザーサイティング】 ※ロックオン後はプラズマキャノンが相手に向かって飛ぶ。

　　　【自動】a(hold) 　　        　※ポインターが相手に重なるとロックオン。
　　　　　　　　　　　　　　　　　　※ロックオン後、再度aを押すと解除。

      【手動】a(hold)→ x　         ※その場をロックオン。
　　　　　　　　　　　　　　　　　　　十字キーでロックオンカーソルを移動出来る。
　　　　　　　　　　　　　　　　　　※ロックオン後、再度aを押すと解除。　
                              
【クローキング】
　　　　　　  　D + a


---------------------------★攻撃系★---------------------------------------
【連打コンボ】
           　y, y, y, y

【ディスク投げ】
 　　　　　　　　　y        ("AvsP" type )
              B(溜め)F + y  ("A" or "B" type )

【プラズマキャノン】
　　　　           x　　   　★空中可

【背後攻撃】　　　　　　　　("AvsP" type のみ)
　　(背後から攻撃されている時)　y

【スライディング】
            　D or DF ＋ z

【サプライズアタック】　※warrior版　　　　　　　　　
【サプライジングロール】※hunter版
    （空中で）D ＋ z
　　
【ダブルキック】　　　　※warrior版
【ライジングアッパー】　※hunter版
　　    　　　D,U ＋ y

【スピア投げ】
　　　　　　　D,DF,F  +  y  ("AvsP" type )
              B(溜め)F + y  ("A" or "B" type )
                            
【空中スピア投げ】
　　　　　　　D,DF,F  +  y

【掴む】
  　　  歩き中に相手に重なる ("AvsP" type )
    (近距離)B,DB,D,DF,F + y  ("A" or "B" type )
               ∟【殴る】　　　            y
　　　　   　　∟【高速連続打撃】          y連打
　 　　    　　∟【移動アッパー】　　      F or B ＋ y　
 　　　    　　∟【グラビティクラッシュ】  jump → y

【刺して放り投げる】
　　　　　　　F,F  +  y 
　　　　　　　　
【自爆】
　          　D,DF,F,D,DF,F ＋ ｓ

　　            ※Lifeが1/4以下の時

【メガクラッシュ】
　　x+y or x+z or y+z

　　　　　　　　※ガード・ヒット問わず敵に当たると最大ライフの16分の1を消費


------------------------------★1000power消費技★--------------------------------------------------
【一定時間エネルギー無限】
　　          D,DF,F,D,DF,F + x

                ※カウントが0になるまでガンゲージが減らない。

------------------------------★3000power消費技★--------------------------------------------------
【スピアをぶん投げる】
　　   D,DF,F,D,DF,F + y

　※ 空中ガード不可。

----------------------------------★細かい点★-----------------------------------------------------
◆ライジングアッパーの1,2段目をプラズマキャノンでキャンセル可能。
◆素手時、レバー前入力しつつ連打コンボの3段目を出すと前方に移動する。
◆グラビティクラッシュは頂点付近から落とすとダメージup
◆フロント・バックダッシュor垂直ジャンプ中のy攻撃は二段目から十字キーでx速度の
　反転ができる(アーケードタイプのみ)
◆空中プラズマキャノンはレバー押し入力で上下前方に撃ち分け可能。
◆メガクラッシュはｘを押しながら別ボタンを押す、もしくはｚを押しながらｘを押しても出る。
◆メガクラッシュはやられ中に出せる。
